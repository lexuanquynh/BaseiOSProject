# BaseiOSProject
The base iOS project build by bazel.

# How to build:
1. Download Python:
  ```
    https://www.python.org/downloads/
  ```

2. Install bazel:
  ```py
    brew install bazelisk
  ```
3. Build:
  ```py
    bazel build //BaseProject:BaseProject  
  ```
4. Run:  
  ```py
    bazel run //BaseProject:BaseProject  
  ```
